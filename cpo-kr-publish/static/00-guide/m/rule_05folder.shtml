HTML 폴더 구조

  1. 현대자동차 : HMC
  2. 제네시스 : GHQ
  3. 폴더명 또는 파일명의 구분을 위해 현대자동차는 H, 제네시스는 G로 구분하여 작업자들 실수를 예방한다.
  4. 가이드 문서 내, 폴더명 또는 파일명 앞에 대문자 H는 현대자동차 기준으로 작성하였다.
    (제네시스는 H를 G로 변경하여 사용한다.)

js 폴더 구조 & 파일명

  1. js 라이브러리는 현대자동차, 제네시스 각 매체별 전체 동일하게 사용한다.
  2. 동일하게 사용하는 라이브러리를 제외하고
    폴더명 또는 파일명의 구분을 위해 현대자동차는 H, 제네시스는 G로 구분하여 작업자들 실수를 예방한다.

CSS 폴더 구조

  1. reset 파일은 현대자동차, 제네시스 각 매체별 동일하게 사용한다.
  2. reset 파일을 제외하고
    폴더명 또는 파일명의 구분을 위해 현대자동차는 H, 제네시스는 G로 구분하여 작업자들 실수를 예방한다.
