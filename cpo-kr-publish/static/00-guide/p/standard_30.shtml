잘못 된 HTML

DOTTYPE

dotype은 html 최상단에 있어야 합니다.


<!DOCTYPE html>
<html lang="ko-KR">
<head>
 
 문서 제목
</head>
<body>
	문서형 정의(DTD:Document Type Definition)은 HTML5, XHTML, HTML의 세가지 문서 유형이 있다.
	기술한 유형에 따라 마크업 문서의 요소와 속성등을 처리하는 기준이 되며 유효성 검사에 이용된다.
</body>
</html>

종료 태그의 처리

HTML5는 종료 태그를 생략할 수도 있습니다. 그러나 모든 요소가 종료 태그를 생략해도 되는 것은 아니기 때문에 종료 태그가 생략할 수 있는 요소를 사전에 확인해야 합니다.
그러나 HTML5에서 종료 태그를 생략하는 것이 가능하다고 하더라도 기존 XHTML1.0의 규칙처럼 시작과 종료 태그를 정확히 명시하여 정형식(Well-Formed) 구조로 마크업할 것을 권장합니다.

<div class="bigSale">
	<ol>
		<li>세일 상품 unit HTML
		div 태그를 닫아줘야 합니다.
		</li>
	</ol>

잘못된 중첩 사용 불가능

시작 태그와 종료 태그의 중첩에 문제가 발생하지 않도록 해야 합니다.

<p><strong>시작 태그와 종료태그 위치는 같아야 합니다.</p></strong>

속성의 중복 사용 불가능

시작 태그에 동일한 속성을 중복하여 선언할 수 없습니다.

<p class="title" class="big_banner">class 속성, style 속성, name 속성 등 두번 선언할 수 없습니다.</p>

ul, ol태그의 바로 아래 자식요소는 li

ol 요소에 올 수 있는 자식 요소는 ul 요소와 마찬가지로 li 요소만 가능합니다.

	<ol>
		<div class="base">
		<li>
			ol 태그와 ul 아래에는 div 태그가 바로 올 수 없습니다.
		</li>
		</div>
	</ol>

	<ul>
		<li>
			<div class="base>
			ol태그와 ul태그 자식 요소인 li 태그 안에 div 태그를 선언해야 합니다.
			</div>
		</li>
	</ol>

tabel 태그 자식 요소

tabel 태그 자식 요소는 caption, colgroup, thead, tfoot, tbody, tr 태그입니다.
tr 태그 자식 요소는 th, td 태그입니다. 

<table border="1">
	<caption>최강 음원 다운로드 (단위:백만)
	<colgroup class="title"></colgroup>
	<colgroup class="singer"></colgroup>
	<colgroup class="download"></colgroup>
	<thead>
		<tr>
			<th scope="col">곡명</th>
			<th scope="col">가수</th>
			<th scope="col">다운로드</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>나와 같다면<</td>
			<td>김연우</td>
			<td>5,896</td>
		</tr>
	</tbody>
</table>
