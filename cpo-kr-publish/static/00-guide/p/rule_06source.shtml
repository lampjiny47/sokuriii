퍼블리싱 개발 환경

로컬 서버 환경

  1. 실행 창에 'appwiz.cpl' 입력하여 제어판의 프로그램 및 기능으로 들어간다.

  2. Windows 기능 켜기/끄기 클릭하여, '인터넷 정보 서비스'를 체크한다.
    html include를 위해 World Wide Web 서비스 > 응용 프로그램 개발 기능 > SSI(Server side include) 를 체크한다.
    확인 버튼을 클릭하여 컴퓨터 다시 시작한다.

  3. 윈도우 > Windows 관리 도구 > IIS인터넷 정보 서비스 관리자를 클릭한다.

  4. Default 사이트의 경로를 변경한다.
