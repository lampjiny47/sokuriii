HMC Swiper

Swiper Option

Banner Swiper를 제외한 옵션 조정 및 추가 기능이 필요할 때 사용하는 Swiper 가이드 입니다.
new Swiper 선언 후 getSwiperOption() 함수의 인자로 옵션값 적용합니다.
추가적으로 swiper 기능 사용시 addOption 값 사용

'getSwiperOption' 함수에서 정의된 옵션
* slidesPerView - view 갯수 {Number}
* spaceBetween - 여백 {Number}
* loop - 반복 {Boolean}
* pagination - bullet {Boolean}
* navigation - next / prev button {Boolean}

swiper-slide의 갯수가 1 이하 일 경우
- pagination : X
- button : X
- spaceBetween : X
- loop : X
- slidesPerView : 1

swiper 고유 API 추가 (Swiper API)
* addOption {} {Object}

  1. 							const swiper1 =  new Swiper('.swiperBanner1',
								getSwiperOption({
									target : '.swiperBanner1', //swiper 사용 할 부모 요소
									slidesPerView : 1, // default : 1
									spaceBetween : 10, // default : 0
									pagination : true, // default : false
									pagination_cls : "01", // default : 01
									navigation : true, // default : false
									loop : true, // default :false
									addOption : { // 추가 옵션 (Swiper API) 
										
										autoplay : 1000,
										on : {
											init : function () {
											  console.log('test');
										},
										
									},
								})
							)
